Question 1 : canversion Fahrenheit - Celsius

Après avoir complété les 2 méthodes de la classe, j'obtiens le résultat demandé.
la méthode float fahrenheitEnCelsius(int f) :
/**
     * la méthode à compléter.
     * 
     * @param f
     *            la valeur en degré Fahrenheit
     * @return la conversion en degré Celsius
     */
    public static float fahrenheitEnCelsius(int f) {
        float celsius = (float)5/9 * (f -32);
        
        return celsius; 
    }

     * Application qui convertit les dégrés Fahrenheit entrés en ligne de commande en dégrés Celsius

     *            ...
     */
