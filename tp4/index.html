Question 1

Vous trouverez ci-dessous les 3 classes de la question 1 complétées.
Classe ConcreteSubject : la classe Observable. Cette classe s'agit d'une liste. A chaque fois qu'on ajoute un nouveau élément à la liste, l'observable notifie l'observateur du changement avec la méthode notifyObservers(). De cette façon, la classe Observer sera au courant des changements dans la classe Observable et pourra mettre à jour le données en conséquence.
package question1;

import java.util.Observable;
import java.util.ArrayList;

/**
 * Décrivez votre classe ConcreteSubject ici.
 * 
 * @author (votre nom)
 * @version (un numéro de version ou une date)
 */
public class ConcreteSubject extends Observable {

    /** ConcreteSubject est composé d'une liste list */
    private ArrayList list;

    public ConcreteSubject() {
        list = new ArrayList();
    }

    public void insert(String name) {
        list.add(name);
        setChanged();
        notifyObservers(name);
    }

    public String toString() {
        return list.toString();
    }

}
